Project #4: Experimental Clock

Design Process

I created an pizza illustration on p5 and then started to experiment with time and IF / ELSE Statement. If you only focus on the pizza, you may feel time is slow since the the illustration only changes every 2 seconds, but after the seeing 2 pizzas dissapear, you may be surprised that only a minute had passed.

Reflection

I'm glad that my code worked and the result turned out as I expected. Overall, it's interesting to learn how to work with time and the IF/ELSE statement.
