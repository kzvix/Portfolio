Project #7: Data Portrait

Description

In this data portrait assignment, I decided to track what I drink in a week. Each row represents a day of the week, starting from top to bottom. "1" represents water, "333" represents coffee, "2" represents soymilk, "55" represents matcha, and "10" represents smoothie.

Design Process

I started by collecting data in my notebook, decided how to represent and arrange them in symbols, and finally recreated it in p5.

Reflection

After this assignment, I realized I need to drink more water, but I'm glad I learned an effective way to visualize data using code.
