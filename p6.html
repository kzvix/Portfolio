Project #6: Optical Illusion

Design Process

For this assignment, I created 3 sketches consist of random sized dots and numbers. These can be used for testing color blindness, which is a strategy that can help determine color vision deficiency. The answer and explnation will be shown if you pressed the mouse on the canvas.

Reflection

I am glad that I was able to relate optical illusion with a diagnostic tool that most the people are familiar with. I started by researching about color blind test and the color combinations that are usually being used. I then experimented with the loop and random functions to create the color blind plates.
